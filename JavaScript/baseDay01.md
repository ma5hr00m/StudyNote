## JavaScript基础

### 注释

```javascript
//单行注释，随便写点东西

/*********************
*多行注释，不知道该写点啥*
*这个注释框没有啥实际用处*
**********************/
```

### 数据类型

- Number数字
  - `123` | `1.5`
- String字符串
  - `"string"` | `"500"`
- Array数组
  - `[12, 2323, 11, 43]`
- Object对象
  - `{key:‘value’,key:'value}`
- Function函数
  - `function myFunction(a, b) { return a * b;}`

### 数据显示

```javascript
window.alert()		//警告框
document.write()	//内容写到.html文件中
innerHTML()			//写入某元素
console.log()		//写入browser控制台
```
